(pull request, patch, or code suggestion) you agree that:**

1. Your contribution is licensed to the project under **GPL-3.0-or-later**, and
2. You grant **Clawnify** a perpetual, worldwide, non-exclusive, royalty-free

This is the inbound-licensing model used by most dual-licensed projects; you
keep the copyright to your contribution.

## Development

See the [README](./README.md) for setup (`bun install`, native module rebuild,
and `bun run --filter @ateam/desktop dev`). Please run `bun test` and
`bun run typecheck` before opening a PR.
